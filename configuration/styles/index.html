Styles

Customize the tour appearance with CSS custom properties (variables).

Visual appearance is fully customizable through CSS custom properties defined on the .rb root element. Override any variable in your own stylesheet to change colors, sizes, spacing, and animations — no need to touch the library source or add extra build steps.
